🎨 AWT Layout Manager Demo

Showcasing BorderLayout, FlowLayout, GridLayout, and CardLayout

About Layout Managers

This demo showcases the four core AWT layout managers implemented in awtea:

  • BorderLayout (top-left) - Arranges components in 5 regions: NORTH, SOUTH, EAST, WEST, and CENTER
  • FlowLayout (top-right) - Arranges components in a left-to-right flow, wrapping to next line when needed
  • GridLayout (bottom-left) - Arranges components in a rectangular grid with equal-sized cells
  • CardLayout (bottom-right) - Shows one component at a time, like a deck of cards

The main window itself uses BorderLayout with a title at NORTH and a grid of demos in CENTER!
