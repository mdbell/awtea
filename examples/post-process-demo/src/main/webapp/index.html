🎨 Post-Processing Demo

WebGL Post-Processing with Bloom Effect

This demo showcases the awtea post-processing pipeline using framebuffer objects (FBOs) for multi-pass rendering. The bloom effect creates a glow around bright objects using threshold extraction, Gaussian blur, and additive blending.

The scene is rendered to an offscreen render target, processed through the bloom pipeline (3 passes), and composited to the screen. The RenderTargetPool efficiently manages FBO allocation.

Keyboard Controls:
B Toggle bloom effect on/off
+ / - Adjust bloom intensity
T Cycle bloom threshold
