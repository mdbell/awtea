Custom Shader Demo

About this example

This example demonstrates the custom shader API for the WebGL rendering backend. It shows a glowing triangle with animated wave distortion and pulsing glow effects.

✓ Custom GLSL vertex and fragment shaders
✓ Time-based wave animation
✓ Pulsing glow effect with color gradients
✓ Direct WebGL rendering via custom geometry
