🗄️ IndexedDB VFS v2 Demo

About This Demo

This demo showcases the IndexedDB Virtual File System v2 implementation, which provides:

✅ Java I/O Compatibility
Use standard java.io.* classes
✅ Large File Support
Handles 100MB+ files efficiently
✅ Lazy Loading
64KB chunks loaded on-demand
✅ Random Access
O(1) seeks with Blob slicing
✅ Performance Stats
Built-in monitoring
✅ Persistent Storage
Files saved in IndexedDB

Demo Sections

  • Basic File Operations: Write and read text files
  • Random Access: Structured data with seeks
  • Large Files: 5MB file with lazy loading
  • Statistics: Performance metrics
