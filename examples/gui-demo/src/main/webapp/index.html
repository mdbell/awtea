awtea GUI Demo

About this example

This comprehensive demo showcases advanced awtea features:

✓ Interactive drawing canvas
✓ Mouse event handling
✓ Graphics primitives (shapes, lines, arcs)
✓ Multiple panels and layouts
✓ Text rendering with various fonts
✓ Color palette demonstration

Try it: Click on the canvas to draw circles, move your mouse to see coordinates!
