awtea Animation Demo

About this example

This comprehensive animation demo showcases real-time physics simulation and interactive controls in awtea. Watch bouncing balls with realistic gravity and collision detection!

Demonstrated Features:

✓ Frame-based animation loop (~60 FPS)
✓ Physics simulation with gravity
✓ Collision detection and bouncing
✓ FPS monitoring and performance tracking
✓ Mouse interaction (click to add balls)
✓ Keyboard controls (pause, reset, effects)
✓ Double buffering for smooth rendering
✓ Motion trails effect (optional)
✓ Velocity vector visualization
✓ Delta-time based physics

Interactive Controls

Mouse Controls:
Left Click - Add ball at cursor
Right Click - Add ball with velocity
Keyboard Controls:
SPACE - Pause/unpause animation
C - Clear all balls
R - Reset to 15 random balls
T - Toggle motion trails
V - Toggle velocity vectors
+ / = - Add one random ball
- - Remove last ball

Physics Details: The simulation applies 400 pixels/second² downward gravity, with a 0.8 damping coefficient on wall bounces for realistic energy loss.
